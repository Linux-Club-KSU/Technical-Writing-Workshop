# Headings
Created using `# text`
## Sub Headings
Created using `## text`
### Sub Sub Headings
Created using `### text`
* Bullet points are created with `* text`
* This is a bullet point
    * This is a sub point
    * It is created using `[tab]* text`
        * This is a sub sub point
            * This can keep going
                * For a while
                    * Without stopping
                        * Some platforms let this go even longer
- Bullet points can also be made with `- text`
- Alternating between `*` and `-` leaves a blank space between them
* Like this
* [x] checkboxes look like `* [x]`
* [ ] checkboxes can be unchecked too `* [ ]`
    * [ ] sub checkboxes work too
1. Lists look like
2. They are made with `1. text`
    1. They can have sub lists too
        1. They can have sub sub lists too

## Formatting
* `**text**` makes it **bold**
* `*text*` makes it *italic*
* `~~text~~` makes it ~~crossed out~~
* Colors can work too `#0969DA`
* `` is used to make `these little text boxes`
* [Links](https://google.com) look like `[text](https://example.com)`
* Images look like `![alt text](image.png)`
